# kakoune-indices

[kakoune](http://kakoune.org) commands to display selections indices in gutter

## Install

Add `indices.kak` to your autoload dir: `~/.config/kak/autoload/`.

## Usage

- `show-indices`
- `hide-indices`
- `enable-indices`
